# Forum

## Overview
React/Python-based app.

## Technological stack
React, MobX, Python (Flask), Webpack, SASS, docker

## Run app
```
npm install
npm run watch
npm run server
```

App will be available at http://localhost:5000.

## Run app in docker
```
docker-compose up
```

App will be available at http://192.168.99.100:5000.
